Research Experience

Chroma 1.0: Real-Time End-to-End Spoken Dialogue Model with Voice Cloning
The first open-source, real-time end-to-end spoken dialogue model that unifies low-latency streaming with high-fidelity voice cloning. Built on a streaming multimodal LLM (Qwen2.5-Omni reasoning + a LLaMA backbone), it reaches 2.3× real-time inference (RTF = 0.43, TTFT = 146.87 ms) and state-of-the-art voice cloning (SIM = 0.817), ranking top-2 on URO-Bench.
